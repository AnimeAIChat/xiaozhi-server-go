Usage notes:
  • Daily per-user quota: 3000 characters, reset at 00:00 each day.
  • For extensive testing, contact the administrator.

Quick Start

  1. Create an agent on the Agents page to bind a device.
  2. In device firmware, set the OTA URL to: https://xiaozhi.xf.bj.cn/api/ota/ and reboot the client.
  3. After reboot, visit the OTA URL to retrieve WebSocket and MQTT+UDP addresses.
  4. Connect to the backend via WS or MQTT (firmware must support MQTT) to obtain a verification code.
  5. Enter the code on the agent page in the console to bind the device.
  6. Reboot the device to start chatting.
